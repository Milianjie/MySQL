## 简单的查询语句DQL

### 语法格式：

```sql
select 字段名1,字段名2,字段名3,...from 表名;//查所有字段用*代替
```

提示：

1、任何一条sql语句以";"结尾，在dos命令窗口中填写命令或语句见";"才能回车执行命令或语句，语句后接\c表示终止语句的填写

2、sql语句不区分大小写

在员工表emp中可以查询到月薪，但能否查询年薪呢？可以

### 由于字段可以参与数学运算，所以可以查询员工的年薪：

```
mysql> select ename,sal*12 from emp;
+--------+----------+
| ename  | sal*12   |
+--------+----------+
| SMITH  |  9600.00 |
| ALLEN  | 19200.00 |
| WARD   | 15000.00 |
| JONES  | 35700.00 |
| MARTIN | 15000.00 |
| BLAKE  | 34200.00 |
| CLARK  | 29400.00 |
| SCOTT  | 36000.00 |
| KING   | 60000.00 |
| TURNER | 18000.00 |
| ADAMS  | 13200.00 |
| JAMES  | 11400.00 |
| FORD   | 36000.00 |
| MILLER | 15600.00 |

```

上面查询出来显示的别名年薪字段为sal*12不好看，可以重写
